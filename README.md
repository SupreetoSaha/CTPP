# CTPP
Chair Table Practice Problem - ZTK

## Controls  
**W/S** - Move fordward/back  
**A/D** - Move left/right  
**Q/E** - Turn left/right  
**R/F** - Look up/down  
**LeftShift/LeftCtrl** - Move up/down  
**Delete** - delete (under mouse pointer)  
**Click** - resize by draging corner, else create new
